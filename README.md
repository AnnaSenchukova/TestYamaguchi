# TestYamaguchi

Адаптивная вёрстка слайдера Hair Styler с использованием GSAP ScrollTrigger.

## Стек

- **Gulp 4** — таскраннер сборки
- **Pug** — препроцессор HTML
- **SCSS** — препроцессор стилей
- **esbuild** — бандлер JS
- **GSAP + ScrollTrigger** — анимации и скролл-триггер
- **BrowserSync** — dev-сервер с hot reload

## Запуск

```bash
npm install
npm run dev
```

Откроется `http://localhost:3000` с hot reload.

## Сборка для продакшена

```bash
npm run build
```

Результат — минифицированная статика в папке `dist/`.
